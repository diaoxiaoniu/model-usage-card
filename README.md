# Model Usage Card

分享 AI 模型使用统计卡片。

## 使用方法

1. 打开 `index.html` 查看卡片
2. 修改 `model-usage-data.json` 更新数据
3. 部署到 GitHub Pages

## 部署到 GitHub Pages

```bash
cd github-pages
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/model-usage-card.git
git push -u origin main
```

然后访问: `https://你的用户名.github.io/model-usage-card/`
